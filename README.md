# Doctor Finder App

A clean Flutter application to find and book doctors.
Built with Riverpod state management and clean architecture.

## Features
- Doctor listing with search
- Doctor detail screen
- Authentication flow

## Tech Stack
- Flutter
- Riverpod
- Clean Architecture
